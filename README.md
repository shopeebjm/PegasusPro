## ЁЯУМ What is Pegasus?

**Pegasus** is a powerful Python-based Android Device Management and Security Audit tool built on top of **ADB (Android Debug Bridge)**. It provides an interactive, color-coded terminal menu to remotely manage, monitor, analyze, and audit Android devices тАФ entirely from your PC, with no third-party app required on the phone.

### Who is it for?

| Audience | Use Case |
|----------|----------|
| ЁЯзСтАНЁЯТ╗ Android Developers | Quick device control and log monitoring during development |
| ЁЯФР Security Researchers | Audit and assess your own device's security posture |
| ЁЯОУ Students | Learn Android internals, ADB commands, and mobile security |
| ЁЯЦея╕П Power Users | Manage and control phones wirelessly over Wi-Fi |

## ЁЯУ▒ Android Device Setup

USB Debugging must be enabled on your Android device before Pegasus can connect.

### Enable USB Debugging

```
Settings тЖТ About Phone
тЖТ Tap "Build Number" 7 times rapidly
тЖТ "You are now a developer!" message appears
тЖТ Go back тЖТ Settings тЖТ Developer Options
тЖТ Toggle ON "USB Debugging"
тЖТ Connect phone to PC via USB
тЖТ On phone popup: tap "Allow"
```

### Switch to Wireless Mode (Wi-Fi ADB)

> One-time USB setup required. After that, go fully wireless.

1. Connect device via USB cable
2. Launch Pegasus тЖТ Select **`[2] Connect a Device`**
3. Enter `y` when asked to enable TCP/IP mode
4. Find your device IP: **Settings тЖТ Wi-Fi тЖТ Tap connected network тЖТ IP Address**
5. Enter the IP in Pegasus when prompted
6. Unplug USB тАФ connection is now wireless тЬЕ
